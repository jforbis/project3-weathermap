Project Purpose:

Weather, climate, and census data is constantly under use by many different enterprises for a multitude of strategic uses, from disaster prediction to construction resource allocation and more. Our model allows a user to value several metrics at once: average rainfall, average temperatures, and relational census data by utilizing an interactive website which allows the user to choose a state and see its capital, high temp, low temp, and precipitation.

The interactivity comes with leveraging leaflet to create cloropleth maps that show each metric and it's gradient from low to high. To accomplish this we did the following:
